# Lumibox — 光の箱パズル

光る箱を、同じ色のゴールへ運ぶ **倉庫番（Sokoban）系のロジックパズル**です。
箱は「押す」ことしかできない（引けない）のがミソ。1マスのミスで詰むので、先を読んで一手ずつ。
スマホ・PC どちらでも快適に遊べます。

🎮 **遊ぶ → https://katoharu0.github.io/lumibox/**

（公開後、GitHub Pages が有効になると上のURLで遊べます）

---

## 遊び方

- **目的**：すべての箱を、対応するゴール（光る枠）に乗せるとクリア。
- **色合わせ**：色つきの箱は **同じ色のゴール** に入れる必要があります（水色・桃色・黄色）。白い箱は白いゴールへ。
- **詰みに注意**：箱は押すだけ。壁ぎわや角に押し込むと動かせなくなることがあります。困ったら **もどす** か **やり直し**。

## 操作

| 端末 | 移動 | その他 |
|------|------|--------|
| スマホ | **スワイプ**（上下左右）／画面下の **Dパッド**／**床をタップ**でその場所まで自動で歩く | 上部ボタンで もどす・やり直し・サウンド |
| PC | **矢印キー** または **WASD**／**床をクリック**で自動移動 | **Z**=もどす、**R**=やり直し、**Esc**=メニュー |

- **MOVES**：今の手数。**BEST**：そのレベルの自己ベスト。
- クリア時、最少手数ぴったりだと **★PERFECT**。手数をちぢめる挑戦も楽しめます。
- 進捗（解放レベル・自己ベスト・サウンド設定）は端末に自動保存されます。

## 特長

- 全15レベル。チュートリアルから色合わせ・複数箱まで、少しずつ難しくなる構成。
- なめらかな移動アニメ／光の演出／クリア時の紙吹雪、効果音（WebAudioで合成、オフ可）。
- 画像・音声ファイル**ゼロ**。`index.html` ひとつで動く軽量1ファイル構成。

## 開発メモ

このゲームは「全レベルが必ず解ける」ことを**機械的に検証**しています。

```bash
# 各レベルが解けるか / 最少手数（PERFECTの基準）を確認
node tools/verify.mjs

# 確認結果（最少手数）を index.html の PARS に書き込む
node tools/verify.mjs --write

# ヘッドレスChromeで実際に起動し、JSエラー・初期化・全レベルのクリア判定を確認
node tools/smoke.mjs
```

- `tools/solver.mjs` … 色合わせ＆デッドマス枝刈り付きの BFS ソルバ（共有モジュール）。
- `tools/verify.mjs` … 全レベルの可解性と最少手数を検証。
- `tools/smoke.mjs` … 実ブラウザでの起動・エラー・クリア判定の総合チェック。

### レベルの記法（`index.html` 内 `LEVELS`）

```
#  壁          (空白) 床
.  ゴール(無色)  $ 箱(無色)   * 箱on ゴール
@  自分         + 自分onゴール
A/B/C/D  色つき箱（色 0/1/2/3）
a/b/c/d  色つきゴール（色 0/1/2/3）
```

レベルを足したら `node tools/verify.mjs --write` を実行すれば、可解性チェックと PERFECT基準の更新が一度に済みます。
